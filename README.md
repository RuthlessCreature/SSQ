# 双色球玄数双模选号

一个本地运行的 Python Web 小程序：

- 自动抓取最近 200 期双色球开奖数据。
- 输入性别、生日类型（公历或农历/阴历）和出生时间后，用子平法排四柱。
- 将八字五行喜用和近期开奖统计合并成“玄学 + 数学”双模型。
- 对历史开奖做滚动回测，迭代多组参数，选出回测评分最高的一组。
- 按页面输入的 `n` 生成 `n` 注号码，并为每注输出理由。

> 彩票开奖是随机事件，任何模型都不能保证中奖。本程序仅用于娱乐、学习和数据实验。

## 运行

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

打开浏览器访问：

```text
http://127.0.0.1:5000
```

## 数据与排盘

- 开奖数据来自中彩网页面使用的 JSONP 接口，程序本地缓存 30 分钟。
- 八字排盘优先使用 `lunar-python`，支持公历和农历输入，按节气换月；如果依赖不可用，会降级为内置近似算法并在页面提示。
- 回测时，每个目标期只使用它之前的历史开奖数据，避免偷看未来数据。
- 生成号码时会避开同一注里红球数字和蓝球数字重复。
