出生信息别到处乱扔。

八字工具看起来轻松，但生日、时间、地点、问题内容拼在一起，就是一份很具体的个人画像。

哪些信息要当回事

  • 完整出生日期和出生时间。
  • 出生地省市区。
  • 具体问事内容，比如感情、工作、财务、健康。

本站怎么处理

排盘在 Cloudflare Worker 里完成。若配置 DeepSeek API，会把结构化盘面和问题发送给模型生成更好读的解释。默认不做用户账号，也不把结果公开成分享页。

你可以怎么保护自己

不要输入身份证号、电话、真实姓名、具体公司名称或他人隐私。问事写到“这份 offer”“这段关系”就够了，不需要把现实世界所有细节都贴上来。
