How to use this guide library

Start with the calculator that matches the main material, then read the related guide before treating the number as a purchase quantity. The calculators give a first-pass estimate; the guides explain why the number may need to change for surface condition, product coverage, waste, pattern direction, trim, or future repairs.

For final buying decisions, check the product label, supplier stock, return policy, dye lot or batch number, and installation instructions. A good early estimate should make the store visit easier, not replace product-specific advice.
